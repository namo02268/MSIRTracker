MSIRTracker

MSIRTracker is an extension block that allows the Mindsensors IR sensor “IRTracker” for SPIKE to be used with EV3 Labs. It supports both A mode and B mode of the infrared ball (RCJ-05) used in RoboCup Junior.

For more details, please check the GitHub repository.

Modes

Measure – AC/DC Direction Mode

Measures the direction of the infrared signal. You can obtain the signal direction of the RoboCup Junior IR ball (A mode / B mode).

Select – AC/DC Direction Mode

A direction-selection mode that can be used in switch blocks. Select “MSIRTracker” → “Measure” → “AC/DC Mode” → “Direction” to use this mode.
